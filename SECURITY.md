# Security Policy

Please report security issues privately and **do not** open a public issue.

- Use GitHub private security reporting: `https://github.com/c4rb0nx1/tuprwre/security/advisories/new`
- Include:
  - Affected version(s)
  - Reproduction steps
  - Impact and severity
  - Any relevant logs or proof of concept

We review and acknowledge reports as soon as possible and coordinate fixes and disclosures with the reporter.
